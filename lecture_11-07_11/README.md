### Usage:
```sh
python3 run.py [OPTIONS]

Options:
    -m, --mode <mode>  Mode of operation [possible values: client, server (default)].
    -o, --host <host>  IPv4 address of the host [default: 127.0.0.1].
    -p, --port <port>  Specifies a port [default: 11111].
    -h, --help         Print help
```
